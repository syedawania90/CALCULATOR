# Simple Calculator Project

This is a simple calculator built using **HTML, CSS, and JavaScript**.  
It performs all basic arithmetic operations: addition (+), subtraction (−), multiplication (×), and division (÷).  

## Features

- Clean and user-friendly interface
- Real-time input and result display
- Clear (C) and Delete (⌫) buttons
- iPhone-style design with rounded corners
- Fully functional using mouse clicks

## Files

- `calculator.html` → Main HTML file
- `style.css` → CSS file for styling (optional if inline CSS)
- `script.js` → JavaScript file for functionality

## How to Use

1. Open `calculator.html` in your browser.
2. Click the buttons to perform calculations.
3. Use `C` to clear the screen, `⌫` to delete last digit, and `=` to calculate the result.
